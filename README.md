# Multiple Image stitching in Python

This repository contains an implementation of multiple image stitching. 

Requirements : 

- Python 2.7
- Numpy >= 1.8 
- OpenCV 3.1.0 


Project Structure : 
	
		|_ code -|
		|		 |-- pano.py
		|		 |-- txtlists-|
		|		 			  |--files1.txt .... 
		|	
		|_ images - |
		|			|- img1.jpg
		|			|- abc.jpg 
		|			.... and so on ... 

Demo txtfile : 
files1.txt :

        ../../images/1.jpg
        ../../images/2.jpg
        ../../images/3.jpg
        ../../images/4.jpg

To run : 

    `python pano.py <txtlists/filename_.txt>`
